## How to View

1. Open `index.html` in any modern web browser
2. Use keyboard arrows to navigate:
   - **Right Arrow** or **Down Arrow** - Next slide
   - **Left Arrow** or **Up Arrow** - Previous slide
   - **Spacebar** - Next slide
   - **Home** - First slide
   - **End** - Last slide
